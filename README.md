## Quick start

```sh
cp manifest/config/config.example.yaml manifest/config/config.yaml
# edit database + paths

```

- Site: http://127.0.0.1:8081
- Admin: http://127.0.0.1:8081/admin

Branches: `develop` (active work), `main` (stable). Version: **5.0.0**.
